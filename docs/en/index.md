## Source Transparency Note

Gaster Code is based on community research and organization of leaked Claude Code source code, with product and development inspiration from the `cc-haha` project. Gaster Code is not an official Anthropic, Claude Code, or `cc-haha` project, and is not affiliated with those projects or organizations.

Starting with **v1.0.0**, this repository is publicly maintained so developers can inspect the source, reproduce builds, submit fixes, and continue improving local AI coding workflows, desktop usability, model integrations, automation, and extension systems.
