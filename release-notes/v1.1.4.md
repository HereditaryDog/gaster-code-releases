# Gaster Code V 1.1.4

V 1.1.4 是 Gaster Code Electron 桌面端设置体验和发布同步版本，重点处理同事公开仓库 PR 的审查移植、设置页二级导航和本轮 macOS/Windows 安装包验证。

## 主要更新

- 设置页改为分组二级侧栏，支持搜索设置项，并将账户、工作区、AI 系统和可观测性入口分组展示。
- 设置页成为当前页面时隐藏全局 TabBar，避免顶部标签栏和设置页二级导航重复抢占视觉层级。
- 设置页默认进入账号页，为后续桌面端注册、登录、充值和订阅体验继续收口。
- 中文设置导航补齐“智能体”“电脑控制”等本地化标签。

## PR 审查

- 移植并修正同事公开仓库 PR #10 `feat: refine settings secondary sidebar`。
- PR #9 `fix: restore sidebar menu glass styling` 基于公开发布仓库旧 `v1.1.2` 基线；原样合入会逆转当前源码 `main` 上的主侧栏 opaque/paint-containment 性能保护，因此本次不直接合入该旧补丁。
- 公开发布仓库 `main` 会随本版本发布同步到新的源码基线，避免继续停留在旧 `v1.1.2` 状态。

## 发布与安装包

| 平台 | 安装包 |
|------|--------|
| macOS Apple Silicon | `Gaster-Code-1.1.4-mac-arm64.dmg` |
| Windows x64 | `Gaster-Code-1.1.4-win-x64.exe` |

本地 Apple Silicon 测试包继续使用内部命名：`Gaster Code-a 1.1.4.dmg`。

macOS 用户如果首次启动提示“已损坏，无法打开”、“此文件已损坏”或“无法验证开发者”，请先将 `Gaster Code.app` 拖入 `Applications`，然后在终端执行：

```bash
xattr -cr /Applications/Gaster\ Code.app
```

如果仍然打不开，可以再执行一次带管理员权限的命令：

```bash
sudo xattr -cr /Applications/Gaster\ Code.app
```

## 验证重点

- 打开设置页，确认左侧二级导航按分组展示，搜索 `mcp` 能过滤到 MCP 设置项。
- 在中文界面下打开设置页，确认“智能体”“电脑控制”等导航标签显示为中文。
- 从普通会话页切到设置页，确认全局 TabBar 隐藏，切回会话页后恢复。
- 打开长上下文会话和侧边栏，确认本轮没有回退 `v1.1.3` 的主侧栏性能保护。
- 使用本地 macOS Apple Silicon DMG 和 GitHub Actions Windows x64 构建产物分别验证安装包结构。
