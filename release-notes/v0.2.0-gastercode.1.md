# Gaster Code V 0.2.0-gastercode.1

这是 Gaster Code 的桌面端升级版本。版本保留 G-Master API 官方登录、默认服务商、品牌和绘图定制，同时带来工作区和桌面体验改进。

## 桌面体验升级

- 新增工作区文件面板，用于浏览文件树、最近变更、代码、Diff 和图片预览。
- 更新诊断、质量门禁和桌面构建流程。
- 改进会话恢复、文件搜索、工具展示和消息渲染。
- 保留 0.2.0 发布所需的桌面版本、Cargo 锁文件和发布工作流兼容性。

## Gaster 定制

- 恢复 G-Master API 网页授权登录和个人账户页。
- 通过 G-Master API 登录的用户默认使用官方 G-Master API 服务商；用户不手动切换时不会回落到 Claude Cloud 原生接入。
- 官方服务商自动继承账号当前分组、API 地址、密钥和可用模型映射。
- 恢复 GPT Image 2 绘图入口，支持提示词优化、尺寸选择、历史缩略图和 5 分钟图片生成等待。
- 输入框动态发光只在模型或 Agent 实际工作时出现，回复结束后自动恢复普通状态。
- 侧边栏恢复 Gaster Code 玻璃样式，并将设置项文案统一为「权限管理」和「远程控制」。
