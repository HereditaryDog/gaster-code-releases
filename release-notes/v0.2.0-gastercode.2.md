# Gaster Code V 0.2.0-gastercode.2

这是 Gaster Code 的修复发布版本。版本继续保留 G-Master API 官方登录、默认服务商和 GPT Image 2 绘图入口，并补齐深色模式、品牌展示和关于页信息。

## 主要变化

- 修复绘图页面在深色主题下仍使用固定浅色背景的问题，输入区、预览区、提示卡片和操作控件现在统一使用主题变量。
- 侧边栏和关于页恢复 G 字 SVG 标识，侧边栏品牌字样恢复为旧版 Gaster Code wordmark 样式。
- 关于页作者改为 HereditaryDog，并删除无关社交媒体入口。
- 关于页完善项目信息区块。
- 更新 README、桌面端安装文档和版本入口文案到 `V 0.2.0-gastercode.2`。

## 安装包

推送 `v0.2.0-gastercode.2` 标签后，GitHub Actions 会自动构建 macOS 和 Windows 桌面安装包，并使用本文件作为 GitHub Release 正文。
