# Gaster Code V 1.0.7

## Highlight

V 1.0.7 合并了桌面端长上下文切换的渲染性能优化、G-Master API 账号与计费中心、发布文档 hygiene 修复，并把桌面端版本推进到 `1.0.7`。

## 主要更新

- 优化桌面端标签页、Markdown 渲染和工作区面板的重复渲染路径，降低长上下文会话切换时的卡顿。
- 内置 G-Master API 账号中心，支持桌面端注册、登录、余额查看、充值、订阅创建、取消、恢复和交易记录。
- 同步官方默认服务商状态，桌面端会根据账号授权和订阅状态刷新可用能力。
- 保留 GPT Image 2 异步图片任务路径，复杂绘图提示词会原样提交，不在桌面端改写或裁剪。

## 问题修复

- 修复长消息内容在会话切换时触发不必要 Markdown 重算的问题。
- 修复工作区面板和标签栏在高频状态变化下的额外渲染。
- 修复 release hygiene 相关文档、依赖锁文件和 docs 构建配置不一致问题。
- 补充账号计费、渲染性能和发布文档相关回归测试。

## 安装说明

本版本安装包会通过桌面发布工作流构建，并同步到公开 release-only 仓库：

- macOS Apple Silicon：`Gaster-Code_1.0.7_macos_arm64_dmg.dmg`
- macOS Intel：`Gaster-Code_1.0.7_macos_x64_dmg.dmg`
- Windows x64：`Gaster-Code_1.0.7_windows_x64_nsis.exe`
- Linux x64：`Gaster-Code_1.0.7_linux_x64_deb.deb`

本地 Mac 测试包仍沿用内部命名：`Gaster Code-a 1.0.7.dmg`。

## 验证重点

- 长上下文会话切换时，页面应先切换，再逐步加载会话内容，操作不应被 Markdown 渲染阻塞。
- 登录、注册、充值、订阅、取消订阅、恢复订阅和交易记录应通过 G-Master API 完成。
- 绘图复杂提示词应原样进入异步图片任务。
- README、安装指南、release notes、桌面版本号和 Tauri 打包版本应保持 `1.0.7` 一致。
