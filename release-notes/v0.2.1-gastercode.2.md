# Gaster Code V 0.2.1-gastercode.2

这是一个 updater 发布链路修复版本。主仓库继续保持私有，桌面端更新器改为读取公开 release-only 仓库中的 `latest.json` 和安装包。

## 更新内容

- 将 Tauri updater endpoint 从私有主仓库切换到公开 release-only 仓库：`HereditaryDog/gaster-code-releases`。
- GitHub Actions 发布流程在主仓库构建完成后，会把 `latest.json`、安装包和签名文件同步到公开 release-only 仓库。
- 文档中的下载地址改为公开 release-only 仓库，避免安装用户无法访问私有 GitHub Release。

## 重要说明

- `0.2.1-gastercode.1` 及更早安装包内置的 updater endpoint 指向私有主仓库，不能通过自动更新发现本版本。
- 请手动安装一次 `0.2.1-gastercode.2`。从这个版本开始，后续更新会通过公开 release-only 仓库检查。
- 主仓库源码仍保持私有；公开仓库只保存安装包、签名文件和 `latest.json`。

## 安装包

- macOS Apple Silicon：`Gaster-Code_0.2.1-gastercode.2_macos_arm64_dmg.dmg`
- macOS Intel：`Gaster-Code_0.2.1-gastercode.2_macos_x64_dmg.dmg`
- Windows x64：`Gaster-Code_0.2.1-gastercode.2_windows_x64_nsis.exe`
