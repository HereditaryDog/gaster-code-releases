# Gaster Code v1.0.0

Gaster Code 1.0.0 是项目公开后的第一个正式开源版本。

## 版本重点

- 将 Gaster Code 确立为独立公开项目，统一包名、桌面端、发布页和 Release 身份。
- 清理公开仓库元数据、Issue 模板、文档链接、About 页面归属信息和旧项目标识。
- 保留桌面端、本地服务、模型服务商配置、IM 适配器、定时任务、Drawing、Skills、Agents 和诊断导出等核心能力。
- 加入发布前品牌扫描，避免旧项目标识重新进入用户可见代码、文档或发布资产。
- 公开源码、安装包、发布说明和 updater metadata，方便社区审查、复现构建和继续二次开发。

## 升级说明

- 桌面端 package、Tauri 元数据和应用内 About 页面版本统一为 `1.0.0`。
- 本地数据迁移会继续保持当前 Gaster Code 配置规范化，并只写入 Gaster Code 自有存储。
- `latest.json` 已指向公开仓库 Release 资产，可用于桌面端自动更新检查。

## 来源说明

Gaster Code 的代码基础来自 Claude Code 泄露源码的社区整理与研究，产品形态和二次开发灵感来自 `cc-haha` 项目。Gaster Code 不是 Anthropic、Claude Code 或 `cc-haha` 的官方项目，也不与上述项目或组织存在官方关联。
