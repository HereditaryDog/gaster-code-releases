# Gaster Code v1.0.0

Gaster Code 1.0.0 is the first public open-source release line for the project.

## Highlights

- Establishes Gaster Code as an independent public project with its own package, desktop app, and release identity.
- Cleans public repository metadata, issue templates, documentation links, and About-page attribution for the new project home.
- Keeps the desktop app, local server, provider settings, IM adapters, scheduled tasks, Drawing, Skills, Agents, and diagnostics workflows available in one release.
- Adds stricter release-readiness checks so legacy branding does not return to user-facing code or release assets.

## Upgrade Notes

- This release uses version `1.0.0` across the desktop package, Tauri metadata, and in-app About page.
- Local data migration support keeps current Gaster Code settings normalized and continues writing only to Gaster Code storage.
