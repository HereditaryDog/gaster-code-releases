# Gaster Code v1.0.4

Gaster Code 1.0.4 is a desktop reliability and long-session performance update for the public source line.

## Highlights

- Improves long conversation switching with measured transcript virtualization, markdown rendering cache reuse, memoized chat blocks, and tab-close cache cleanup.
- Fixes streaming tool-call and Ask User Question state handling so active sessions keep pending interactive state visible during reconnects.
- Broadcasts session stream updates to multiple connected desktop clients without dropping Gaster/G-Master-specific session behavior.
- Adds H5 LAN diagnostics, stale-host detection, and session branching support for the desktop conversation view.
- Stabilizes Computer Use dependency setup with stricter Python version checks, Pillow compatibility pinning, and pip mirror fallback.
- Improves workspace/code rendering reliability, registered filesystem roots, and the project context chip.
- Improves Telegram adapter streaming output and markdown table formatting for remote conversations.
- Adds configurable AI request timeout and manual proxy settings for desktop/server network requests.
- Aligns root package metadata, desktop package metadata, Tauri metadata, Rust package metadata, lockfiles, and the About page display version to `1.0.4`.

## Upgrade Notes

- No user data migration is required.
- Current public source version is `1.0.4`.
- Local Apple Silicon test packages use the `Gaster Code-a 1.0.4.dmg` naming convention.
- Current downloadable public installer assets may remain on the latest successful public release until the `v1.0.4` release workflow completes signing, packaging, and asset publication.
