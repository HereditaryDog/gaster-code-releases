# Gaster Code V 0.2.3-gastercode.1

这是一次 Gaster Code 桌面端能力更新。重点是补齐 H5 手机访问、移动端聊天体验、活动统计和多项桌面运行时修复，同时继续保留 Gaster Code 品牌、G-Master 服务商流程、精选 Skills/Agents 和旧配置兼容迁移。

## 更新内容

- 新增 H5 访问能力：支持设置页开启、生成一次性 Token、配置允许来源、填写公开访问地址，并为远程 REST/WebSocket 请求启用 Token 校验。
- 优化手机浏览器体验：窄屏优先显示聊天主界面，侧栏改为抽屉，模型、权限、项目、分支和 worktree 选择器改为底部弹层。
- 新增本地用量 / 活动统计页，展示会话、消息、工具调用、模型 Token 和每日活动趋势。
- 新增回复语言设置，允许模型回复语言独立于界面语言配置。
- Computer Use 支持自定义 Python 解释器，并在解释器变化后重建运行环境。
- 改进会话标题清理、context token 展示、定时任务 markdown summary、Agent 缓存刷新和 worktree 启动状态去重。
- 修复 AskUserQuestion 多选 marker、provider fallback 缺失 `usage`、定时任务 CLI entrypoint 继承过期环境等问题。
- 更新文档与 H5 访问说明，继续清理非必要的旧版用户可见残留。
