# Gaster Code V 0.2.1-gastercode.6

这是一个 G-Master 统一账号入口版本，重点补齐桌面端登录与注册分流，让新用户可以直接从 Gaster Code 进入 G-Master API 注册流程，注册完成后继续回到桌面授权链路。

## 更新内容

- 桌面端个人设置页新增「注册 G-Master 账号」入口，与「使用 G-Master API 登录」共享同一套授权回调和本地服务商同步流程。
- 本地 `/api/gmaster-auth/start` 支持 `login` / `register` intent；缺省仍按登录处理，非法 intent 返回 400。
- 服务端授权会话记录 intent，便于后续排查桌面端发起的是登录还是注册流程。
- 浏览器开发模式下预留授权弹窗，修复异步获取注册链接后被浏览器拦截导致点击注册没有反应的问题。
- 发布矩阵在 tag 自动发布时包含 Windows x64，和 macOS ARM64、macOS x64、Linux x64 一起构建并同步到公开下载仓库。
