# Gaster Code V 0.2.1-gastercode.3

这是一个 IM adapter 品牌修复版本，适合从 `0.2.1-gastercode.2` 通过桌面端更新器直接测试升级。

## 更新内容

- 修复微信配对流程中仍提示 `Claude Code` / `Claude` 的问题，现在未授权和配对成功文案都会显示 `Gaster Code`。
- 同步修复 Telegram、飞书、钉钉 adapter 的配对提示，统一通过共享 formatter 输出品牌文案。
- 将钉钉权限卡标题和普通消息标题改为 `Gaster Code`。
- 更新 IM adapter 文档，避免运行时说明继续引用旧品牌。

## 验证

- `cd adapters && bun test`
- `bun run check:adapters`

## 安装包

- macOS Apple Silicon：`Gaster-Code_0.2.1-gastercode.3_macos_arm64_dmg.dmg`
- macOS Intel：`Gaster-Code_0.2.1-gastercode.3_macos_x64_dmg.dmg`
- Windows x64：`Gaster-Code_0.2.1-gastercode.3_windows_x64_nsis.exe`
