# Gaster Code V 0.2.0-gastercode.3

这是 Gaster Code 桌面端自动更新修复发布版本。重点修复 GitHub Release 缺少 Tauri updater manifest 的问题，让已安装的桌面端可以通过「设置 -> 关于 -> 检查更新」检测并安装新版本。

## 主要变化

- 修复 release CI 覆盖 `createUpdaterArtifacts` 为 `false` 的问题，正式发布会重新生成 Tauri updater 所需的 `latest.json` 和签名产物。
- 新增发布配置回归测试，避免后续发布再次关闭 updater artifacts。
- 关于页版本 fallback 改为读取统一版本常量，避免 Web UI 或测试环境展示旧版本号。
- 保留本轮桌面端体验调整：Codex 风格深色配色、绘图页深色模式适配、侧边栏 GitHub 入口指向作者主页，以及 G-Master API 模型组置顶。
- 更新 README、桌面端安装文档和版本入口文案到 `V 0.2.0-gastercode.3`。

## 安装包

推送 `v0.2.0-gastercode.3` 标签后，GitHub Actions 会自动构建 macOS、Linux 和 Windows 桌面安装包，并使用本文件作为 GitHub Release 正文。

发布完成后，Release 资产中应包含 `latest.json`。现有 `0.2.0-gastercode.2` 安装包会通过这个 manifest 检测到本版本更新。
