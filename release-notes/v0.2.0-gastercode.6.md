# Gaster Code V 0.2.0-gastercode.6

这是 Gaster Code 桌面端品牌与服务商显示修复版本。本轮主要面向 Windows 端安装包复验与修复；macOS 安装包会跟随同一个 tag 同步构建，不单独拆维护分支。

## 平台说明

- Windows：本轮主验证目标，发布 `Gaster-Code_0.2.0-gastercode.6_windows_x64_nsis.exe`。
- macOS：同版本同步产物，发布 Apple Silicon 与 Intel 两个 `.dmg`。

## 更新内容

- 修复新建会话空态仍显示旧项目图标的问题，统一使用 Gaster Code 的 G 标识。
- 修复新建会话说明和输入框占位文案中的 `Claude` 残留。
- G-Master API 官方服务商现在会一直显示在「设置 -> 服务商」页面。
- 未登录 G-Master API 时，官方服务商卡片保持暗色边框并禁用测试/编辑操作。
- 登录 G-Master API 后，官方服务商卡片边框会亮起；是否显示「默认」徽标仍只取决于它是否为当前默认服务商。

## 发布验证

- `cd desktop && bun run lint`
- `cd desktop && bun x vitest run src/__tests__/generalSettings.test.tsx src/__tests__/pages.test.tsx --exclude "build-artifacts/**"`
