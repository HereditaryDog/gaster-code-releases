# Gaster Code v1.0.1

This maintenance release fixes Drawing reliability for complex GPT Image 2 prompts.

## Fixed

- Drawing now creates a G-Master API async image job and polls the job result instead of holding a single long image-generation request open.
- The image-generation wait window is extended to 15 minutes so complex prompts have time to finish without hitting the old client-side timeout path.
- User prompts are sent exactly as entered for image generation. Gaster Code no longer trims, compresses, shortens, or rewrites the prompt before submitting it.
- Upstream 504/524 and interrupted image-channel failures now show stable error messages without telling users to simplify the prompt.
