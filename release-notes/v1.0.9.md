# Gaster Code V 1.0.9

## Highlight

V 1.0.9 是一个桌面端修复版本，重点收口 G-Master 登录授权版本号和 GPT Image 2 异步绘图轮询稳定性。安装包含本版本的客户端后，新的 G-Master OAuth 授权页和登录管理记录会显示当前桌面版本，而不是历史 fallback `0.1.3`。

## 主要更新

- G-Master OAuth 授权启动改为直接读取桌面端打包版本号，避免打包环境没有 `VITE_APP_VERSION` 时回退到旧版本。
- GPT Image 2 异步绘图任务创建成功后，轮询阶段遇到网络中断、504 或 524 临时超时时会有限重试。
- 图片轮询重试不会重新提交 `/v1/images/generations/async` 生成请求，避免同一个提示词被重复创建任务。

## 问题修复

- 修复 G-Master OAuth 授权页和后台登录管理页中，客户端版本长期显示 `0.1.3` 的问题。
- 修复异步图片任务已经创建后，轮询阶段单次网络中断会直接让绘图失败的问题。
- 修复 G-Master 图片通道短暂返回 524/504 时，桌面端无法等待后续任务完成的问题。

## 安装说明

本版本安装包会通过桌面发布工作流构建，并同步到公开 release-only 仓库：

- macOS Apple Silicon：`Gaster-Code_1.0.9_macos_arm64_dmg.dmg`
- macOS Intel：`Gaster-Code_1.0.9_macos_x64_dmg.dmg`
- Windows x64：`Gaster-Code_1.0.9_windows_x64_nsis.exe`
- Linux x64：`Gaster-Code_1.0.9_linux_x64_deb.deb`

本地 Mac 测试包仍沿用内部命名：`Gaster Code-a 1.0.9.dmg`。

## 验证重点

- 新安装或更新到 V 1.0.9 后，重新发起 G-Master 登录/注册授权，授权页和新的管理记录应显示 `1.0.9`。
- 已经存在的历史授权记录不会被桌面端回写，旧记录仍可能保留当时写入的版本。
- GPT Image 2 异步绘图在任务创建成功后，如果第一次轮询遇到临时中断或 524/504，应继续轮询并在任务成功后返回图片。
- README、安装指南、release notes、桌面版本号和 Tauri 打包版本应保持 `1.0.9` 一致。
