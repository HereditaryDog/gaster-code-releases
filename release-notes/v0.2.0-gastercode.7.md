# Gaster Code V 0.2.0-gastercode.7

这是 Gaster Code 首个公开预览候选版本。本版本接续 `0.2.0-gastercode.6` 已验证的多平台安装包与 updater 发布链路，把最新的蓝色品牌主题、Gaster Code 文案清理和文档发布修复纳入正式 release。

## 平台说明

- Windows：发布 `Gaster-Code_0.2.0-gastercode.7_windows_x64_nsis.exe`。
- macOS：同版本同步产物，发布 Apple Silicon 与 Intel 两个 `.dmg`。
- Linux：同版本同步产物，发布 x64 与 arm64 `.deb`。

## 更新内容

- 桌面端主题强调色从旧橙色切换为 Codex 风格蓝色。
- 统一主色、焦点环、选中态、模型选项、主按钮和代码高亮中的品牌强调色 token。
- 清理对话输入框、团队输入区和原型页面中的旧 `Claude` 占位文案，统一改为 Gaster Code。
- 新增主题 token 回归测试，避免旧橙色品牌色重新进入桌面样式。
- 文档更新到 `V 0.2.0-gastercode.7`，安装包命名、README 和桌面端安装指南保持一致。
- 修复 GitHub Pages 文档 workflow 在 Pages 不可用或未启用时直接失败的问题；现在会先完成文档构建，只有 Pages 可用时才执行部署。

## 发布验证

- `cd desktop && bun run test src/__tests__/themeTokens.test.ts src/__tests__/pages.test.tsx -- --run`
- `cd desktop && bun run lint`
- `bun run docs:build`
- `cd desktop && bun run build`

## 已知说明

- macOS 安装包暂未进行 Apple Developer ID 公证，首次打开可能需要右键打开或移除隔离属性。
- Windows 安装包暂未进行代码签名，首次运行可能出现 SmartScreen 提示。
