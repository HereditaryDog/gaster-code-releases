# Gaster Code V 0.2.2-gastercode.1

这是 Gaster Code 的桌面端能力更新版本，同时加入 GasterCode 精选能力 MVP。重点是补齐桌面端仓库启动、运行时隔离、诊断恢复和内置能力开关，并继续保持 GasterCode 品牌与旧配置兼容迁移。

## 更新内容

- 新增桌面仓库启动能力，支持从最近项目和空会话入口选择当前分支、目标分支或隔离 worktree 启动会话。
- 强化服务商运行时隔离，避免桌面端会话、定时任务和标题生成链路串用过期 provider、model 或 thinking 配置。
- 新增安全 doctor 恢复能力，可报告并隔离损坏的本地持久化文件，降低配置损坏导致应用无法启动的风险。
- 增加 Computer Use 授权与禁用控制，保留默认开启体验，同时允许用户在设置中关闭相关能力。
- 改进定时任务桌面通知跳转、会话标题同步、临时目录项目恢复和缺失工作目录诊断。
- 加入 GasterCode 官方精选能力 MVP：内置 10 个 Skills 和 6 个 Agents，安装后默认启用，用户可在设置页自行关闭。
- `/` 命令面板补齐中文说明，减少用户可见英文残留。
- 保留旧版配置目录、旧版环境变量和旧 localStorage key 的兼容读取与迁移，同时优先写入 GasterCode 新命名。
